# ttmihara/dotfiles

## Installation

### Windows

To install [`scoop`](https://github.com/ScoopInstaller/Scoop)
```powershell
iwr -useb get.scoop.sh | iex
```

To install [`chezmoi`](https://github.com/twpayne/chezmoi).
```
scoop install chezmoi
```

To setup this repo
```
chezmoi init ttmihara
```
